# Alertes
## Configuration
```json
'flexible-alerts': {

        style: 'flat',
        note: {
          label: "Information"
        },
        tip: {
          label: "Conseil"
        },
        warning: {
          label: "Avertissement"
        },
        attention: {
          label: "Attention"
        }
      }
```
## Ecriture
> [!NOTE]
> An alert of type 'note' using global style 'callout'.

> [!TIP]
> An alert of type 'tip' using global style 'callout'.

> [!WARNING]
> An alert of type 'warning' using global style 'callout'.

> [!ATTENTION]
> An alert of type 'attention' using global style 'callout'.
